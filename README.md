# App Review Sentiment Analysis

## 📌 Overview
This project focuses on collecting user reviews from mobile applications using web scraping techniques and analyzing user feedback.

## 🎯 Objectives
- Collect real-world user reviews
- Build a dataset from scratch
- Explore and analyze user feedback

## 📊 Dataset
The dataset is collected from online platforms using automated scraping techniques and includes:
- Review text
- Rating
- App name
- Date

## ⚙️ Methodology
1. Data collection using web scraping
2. Data preparation
3. Analysis of user reviews

## 🚀 How to Run

```bash
pip install -r requirements.txt
python src/scraping.py
